## 🤖 ส่วนที่ 1: 5 Automated Response Scripts

### Automation 1: Price Comment (คอมเมนต์ถามราคา)
**สถานการณ์:** มีคนคอมเมนต์ "ราคาเท่าไหร่ครับ/ค่ะ" ภายใต้โพสต์

**คำตอบอัตโนมัติ:**
```
จีจี้ล้อซิ่งจัดให้ครับพี่! 🔥

ผมส่งรายละเอียดราคาคอมโบเซ็ตสุดคุ้ม 
พร้อมรูปตัวอย่างฟิตเมนท์ตรงรุ่นรถของพี่ 
ไปทางอินบ็อกซ์เรียบร้อยครับ 

เช็กข้อความด่วนเลยครับพี่! 👇

#จีจี้ล้อซิ่ง #บางแวก #ล้อแม็ก
```

**การตั้งค่า:**
- Facebook Page: Settings > Messaging > Instant Replies
- Instagram: Settings > Business > Quick Replies
- ใช้ ManyChat หรือ Chatfuel สำหรับ automation ขั้นสูง

**สิ่งที่ส่งใน Inbox:**
1. ราคาสินค้าที่ถาม
2. รูปตัวอย่างติดตั้งรถรุ่นเดียวกัน
3. โปรโมชั่นปัจจุบัน
4. CTA: "สนใจนัดวันติดตั้งไหมครับ?"

### Automation 2: Location Comment (คอมเมนต์ถามที่ตั้ง)
**สถานการณ์:** มีคนคอมเมนต์ "ร้านอยู่ที่ไหนครับ", "ขอพิกัดหน่อย"

**คำตอบอัตโนมัติ:**
```
ร้านจีจี้ล้อซิ่ง อยู่บนถนนบางแวก พุทธมณฑลสาย 2 ครับ 
เดินทางง่ายมาก! 🗺️

ผมส่งพิกัดแผนที่ Google Maps นำทาง 
ไปให้ในแชทแล้วนะครับ 

แวะมาชมล้อแม็กจริงหน้าร้านได้เลยครับพี่! 
เปิดทุกวัน 9:00-18:00 ครับ 🙏

#จีจี้ล้อซิ่ง #บางแวก #ล้อแม็ก
```

**สิ่งที่ส่งใน Inbox:**
```
📍 พิกัดร้าน:
https://goo.gl/maps/[link]

🚗 การเดินทาง:
- จาก BTS บางหว้า: 10 นาที
- จากทางด่วนพุทธมณฑล: 5 นาที
- ที่จอดรถกว้างขวาง

⏰ เวลาเปิด-ปิด:
จันทร์-อาทิตย์: 9:00-18:00

📞 ติดต่อ:
โทร: [เบอร์โทรศัพท์]
Line: [Line ID]
```

### Automation 3: Fitment Comment (คอมเมนต์ถามฟิตเมนท์)
**สถานการณ์:** มีคนคอมเมนต์ "รุ่นนี้ใส่ได้ไหมครับ", "ติดซุ้มไหม"

**คำตอบอัตโนมัติ:**
```
รุ่นนี้จัดทรงให้จบแบบไม่ติดซุ้ม ไม่ล้อยื่นแน่นอนครับพี่! ✅

ช่างจีจี้ส่งสเปกออฟเซ็ตและขนาดยาง 
ที่ใส่แล้วเป๊ะพอดีไปให้ในข้อความแล้วครับ 

ลองเช็กดูได้เลยครับพี่! 
ถ้ามีคำถามเพิ่มเติม ทักมาได้เลยนะครับ 🙏

#จีจี้ล้อซิ่ง #จัดทรง #ฟิตเมนท์
```

**สิ่งที่ส่งใน Inbox:**
1. สเปกแนะนำสำหรับรุ่นรถนั้น
2. รูปตัวอย่างรถรุ่นเดียวกันที่ติดตั้งแล้ว
3. ราคาประมาณการ
4. คำถามเพิ่มเติม: "พี่สะดวกแวะมาให้ช่างวัดระยะไหมครับ? ฟรี!"

### Automation 4: Finance Comment (คอมเมนต์ถามผ่อน)
**สถานการณ์:** มีคนคอมเมนต์ "มีผ่อนไหมครับ", "ผ่อน 0% ไหม"

**คำตอบอัตโนมัติ:**
```
มีโปรผ่อนสบายกระเป๋า 0% แน่นอนครับพี่! 💳

จีจี้ส่งรายละเอียดบัตรเครดิตที่ร่วมรายการ 
และตารางยอดผ่อนต่อเดือนสำหรับชุดคอมโบล้อ+ยาง 
ไปให้ทางอินบ็อกซ์แล้วนะครับ

เช็กได้เลยครับ ผ่อนเริ่มต้นเพียงเดือนละ X,XXX บาทเท่านั้น! 🔥

#จีจี้ล้อซิ่ง #ผ่อน0percent #ล้อแม็ก
```

**สิ่งที่ส่งใน Inbox:**
```
💳 บัตรเครดิตที่ร่วมรายการ:
- KTC
- Krungsri
- Bangkok Bank
- SCB
- และอื่นๆ

📊 ตัวอย่างการผ่อน:
ชุดล้อ + ยาง 15,900 บาท
ผ่อน 0% 10 เดือน = เดือนละ 1,590 บาท

✅ เอกสารที่ใช้:
- บัตรประชาชน
- สำเนาทะเบียนรถ
- Statement 3 เดือน (บางธนาคาร)

สนใจให้ผมช่วยคำนวณยอดผ่อนให้ไหมครับ?
บอกรุ่นรถและงบประมาณมาได้เลยครับ! 🙏
```

### Automation 5: Shipping Comment (คอมเมนต์ถามจัดส่ง)
**สถานการณ์:** มีคนคอมเมนต์ "ส่งต่างจังหวัดไหมครับ", "อยู่เชียงใหม่สั่งได้ไหม"

**คำตอบอัตโนมัติ:**
```
ต่างจังหวัดส่งจริง แพ็กแน่นหนา ถึงหน้าบ้าน 100% ทั่วไทยครับพี่! 📦

ผมส่งรีวิวงานแพ็กของและอัตราค่าจัดส่งด่วนพิเศษ 
ไปให้ในข้อความเรียบร้อยครับ 

สั่งซื้อออนไลน์ได้เลยครับพี่! 
มีบริการถ่วงล้อ+เติมลมให้เสร็จก่อนส่งด้วยครับ 🔥

#จีจี้ล้อซิ่ง #ส่งทั่วประเทศ #ออนไลน์
```

**สิ่งที่ส่งใน Inbox:**
```
📦 กระบวนการจัดส่ง:

1. พี่เลือกสเปกที่ต้องการ
2. เราถ่วงล้อ+เติมลมไนโตรเจน
3. แพ็กกล่อง double-wall + bubble wrap หนา
4. ส่งผ่าน Flash Express / Kerry
5. 2-3 วันถึงหน้าบ้าน!

💰 ค่าจัดส่ง (ประมาณการ):
- กรุงเทพฯ ปริมณฑล: 150-200 บาท
- ภาคกลาง: 250-350 บาท
- ภาคเหนือ/อีสาน: 400-500 บาท
- ภาคใต้: 500-600 บาท

🎁 พิเศษ!
ออเดอร์วันนี้ ส่งฟรี! (ประหยัด XXX บาท)

📸 รีวิวงานแพ็ก:
[ส่งรูปงานแพ็กจริง 2-3 รูป]

สนใจรุ่นไหน บอกมาได้เลยครับ เดี๋ยวผมคำนวณค่าส่งให้! 🙏
```

## ⚠️ ส่วนที่ 2: 5 Crisis Management Protocols

### Crisis 1: Scratched Rim Claim (ลูกค้าแจ้งล้อเป็นรอย)
**สถานการณ์:** ลูกค้าทักมาบอกว่าล้อเป็นรอยระหว่างติดตั้ง

**ขั้นตอนการจัดการ:**

**Step 1: ตอบกลับทันที (ภายใน 30 นาที)**
```
จีจี้ล้อซิ่งกราบขออภัยในความผิดพลาดครับพี่! 😰

ทางร้านยินดีรับผิดชอบเปลี่ยนวงใหม่แกะกล่องให้ทันที 
โดยไม่มีค่าใช้จ่ายครับ 

รบกวนพี่นัดวันเดี๋ยวผมล็อกคิวช่อง VIP 
จัดการให้จบเนียนที่สุดครับ!

พี่สะดวกวันไหนครับ? 
ผมจะดูแลเรื่องนี้ด้วยตัวเองเลยครับ 🙏
```

**Step 2: เมื่อลูกค้ามาถึงร้าน**
- ตรวจสอบรอยด้วยกัน
- ถ่ายรูปร่วมกันเป็นหลักฐาน
- เสนอทางเลือก:
  1. เปลี่ยนล้อใหม่ทันที (ถ้ามีสต็อก)
  2. สั่งจากโกดัง (3-5 วัน) + ให้ล้อชั่วคราวใช้
  3. คืนเงินเต็มจำนวน + ของชดเชย

**Step 3: ติดตามผล**
- หลังแก้ไข ทักถามความพึงพอใจ
- เสนอส่วนลด 10% สำหรับครั้งต่อไป
- บันทึกเป็น case study ป้องกันซ้ำ

**ต้นทุนที่คาดการณ์:**
- ล้อใหม่: ตามราคาทุน
- เสียชื่อเสียง: ถ้าจัดการไม่ดี

---

### Crisis 2: Rubbing Fender Issue (ลูกค้าแจ้งติดซุ้ม)
**สถานการณ์:** ลูกค้าขับไปแล้วแจ้งว่าล้อยึดติดซุ้ม

**ขั้นตอนการจัดการ:**

**Step 1: ตอบกลับทันที**
```
ต้องขออภัยด้วยครับ ช่างเราจัดทรงแน่นไปนิดนึง! 😅

พี่ขับรถกลับเข้ามาที่สาขาบางแวกได้เลยครับ 
เดี๋ยวผมสั่งปรับแต่งสเปกปาดซุ้มหรือเสริมความสูง 
ให้ฟิตเมนท์เป๊ะแบบไม่ติด 100% ฟรีครับพี่!

พี่ว่างวันไหนครับ? 
ผมเตรียมทีมช่างไว้รอเลยครับ 🙏
```

**Step 2: เมื่อลูกค้ามาถึง**
- ตรวจสอบจุดที่ติด
- เสนอวิธีแก้:
  1. ปาดซุ้ม (ถ้าติดเล็กน้อย)
  2. ปรับช่วงล่างเพิ่ม (ถ้าเตี้ยเกินไป)
  3. เปลี่ยนล้อ offset ใหม่ (ถ้าผิดสเปกจริงๆ)
- ทำทันที ให้ลูกค้ารอดูผลลัพธ์

**Step 3: ชดเชย**
- ฟรี บริการทั้งหมด
- ฟรี เติมลมไนโตรเจน
- ส่วนลด 500 บาท สำหรับครั้งต่อไป

**ป้องกันในอนาคต:**
- ถ่ายรูปรถลูกค้าก่อน-หลังทุกครั้ง
- ให้ลูกค้าทดลองขับรอบอู่ก่อนรับรถ
- มีใบรับรองฟิตเมนท์เซ็นร่วมกัน

---

### Crisis 3: Provincial Shipping Delay (จัดส่งล่าช้า)
**สถานการณ์:** ลูกค้าต่างจังหวัดทักมาว่าพัสดุยังไม่ถึง

**ขั้นตอนการจัดการ:**

**Step 1: ตอบกลับทันที**
```
จีจี้ล้อซิ่งต้องขออภัยในความล่าช้าของระบบขนส่งด้วยครับพี่! 😰

ทางเราทำการเช็กสถานะล่าสุด 
พัสดุถึงจังหวัดพี่เรียบร้อยแล้ว 
จะนำจ่ายถึงหน้าบ้านวันพรุ่งนี้แน่นอนครับ 

จะติดตามให้จนกว่าพี่จะได้รับของครับ!
ขอบพระคุณที่มั่นใจสั่งของกับเราครับ 🙏
```

**Step 2: ดำเนินการ**
- ติดต่อขนส่งทันที (Flash/Kerry)
- ขอ tracking number อัปเดต
- ส่ง screenshot ให้ลูกค้า
- โทรติดตามทุก 2-3 ชั่วโมง

**Step 3: ชดเชย**
- หากเกิน 5 วัน: ส่งฟรี下次
- หากเกิน 7 วัน: คืนค่าส่ง + ของแถม
- ส่งข้อความขอโทษพร้อม voucher

**Template ติดตาม:**
```
พี่ครับ อัปเดตพัสดุครับ:
- วันนี้ [วันที่]: ถึงศูนย์กระจาย [จังหวัด]
- คาดการณ์นำจ่าย: พรุ่งนี้ 10:00-14:00

ผมโทรติดตามให้แล้วครับ 
ได้รับยืนยันว่าจะนำจ่ายแน่นอนครับ! 🙏
```

### Crisis 4: Negative Online Review Response (รีวิวเชิงลบ)
**สถานการณ์:** มีรีวิว 1-2 ดาวบน Google/Facebook

**ขั้นตอนการจัดการ:**

**Step 1: ตอบกลับสาธารณะ (ภายใน 2 ชม.)**
```
ทางร้านน้อมรับคำติชมเพื่อปรับปรุงบริการให้ดียิ่งขึ้นครับ 
ต้องขออภัยที่ทำให้พี่ไม่พอใจครับ 😔

ทีมงานจีจี้ได้ทำการทักข้อความหลังไมค์ไปหาพี่ 
เพื่อดูแลและชดเชยค่าเสียหายทั้งหมดเรียบร้อยครับ 

มั่นใจได้เลยว่าจีจี้รับผิดชอบดูแลลูกค้าทุกลำครับ 
หากพี่ไม่ได้รับข้อความ รบกวนทักกลับมาอีกครั้งนะครับ 🙏
```

**Step 2: ติดต่อส่วนตัว**
- ทักหาลูกค้าทันที
- ถามรายละเอียดปัญหา
- เสนอวิธีแก้ + ชดเชย
- ขอโอกาสแก้ไข

**Step 3: ติดตาม**
- หลังแก้ไข: ขอให้ลูกค้าอัปเดตรีวิว (อย่าขอกับให้ลบ)
- บันทึกปัญหา เพื่อป้องกันซ้ำ
- ฝึกอบรมพนักงาน (ถ้าเป็น human error)

**สิ่งที่ห้ามทำ:**
- ❌ โต้เถียงสาธารณะ
- ❌ กล่าวโทษลูกค้า
- ❌ เพิกเฉย
- ❌ ลบรีวิว (ถ้าทำได้)

---

### Crisis 5: Wrong Fitment Delivered (ส่งสินค้าผิดสเปก)
**สถานการณ์:** ลูกค้าได้รับล้อผิดขนาด/ผิดสี

**ขั้นตอนการจัดการ:**

**Step 1: ตอบกลับทันที**
```
เกิดข้อผิดพลาดในการคัดสต็อกขอบล้อ 
ต้องขออภัยพี่อย่างสูงครับ! 😰

เดี๋ยวทางร้านส่งแม็กชุดสเปกเทพตัวตรงรุ่น 
สลับเปลี่ยนพัสดุด่วนที่สุดให้วันนี้เลยครับ 

พี่ไม่ต้องเสียค่าส่งกลับสักบาทครับ!
ทางเราจะส่ง Kerry แบบ door-to-door ให้เองครับ 🙏
```

**Step 2: ดำเนินการ**
- จัดเตรียมของถูกต้องทันที
- นัดรับของผิดจากลูกค้า (Kerry รับถึงบ้าน)
- ส่งของถูกแบบด่วนพิเศษ (overnight ถ้าเป็นไปได้)
- แจ้ง tracking number ทั้ง 2 ทาง

**Step 3: ชดเชย**
- ฟรี ค่าจัดส่งทั้งไป-กลับ
- ของแถมมูลค่า 500-1,000 บาท
- ส่วนลด 10% ครั้งต่อไป
- โทรขอโทษโดยตรง (เจ้าของร้าน)

**Template แจ้งลูกค้า:**
```
พี่ครับ ของถูกต้องส่งออกไปแล้วครับ!

📦 Tracking: [เลขพัสดุ]
🚚 คาดการณ์ถึง: พรุ่งนี้ 10:00-12:00
📞 ขนส่งจะโทรนัดรับของผิด: วันนี้บ่าย

ผมขอโทษจริงๆ ครับ 
ครั้งนี้เป็นบทเรียนให้เราปรับปรุงระบบครับ 
ขอบคุณพี่ที่ให้โอกาสแก้ไขนะครับ 🙏
```

**ป้องกันในอนาคต:**
- ถ่ายรูปก่อนแพ็ก ส่งให้ลูกค้า confirm
- มี checklist ตรวจก่อนปิดกล่อง
- แยกโซนสต็อกตามสเปกชัดเจน

### ManyChat/Chatbot Setup:

**Flow 1: New Subscriber**
```
Trigger: User subscribes
→ Send Welcome Message
→ Ask: สนใจอะไรครับ? (Buttons: ล้อแม็ก/ยาง/ตั้งศูนย์/อื่นๆ)
→ Route to appropriate flow
```

**Flow 2: Price Inquiry**
```
Trigger: Keyword "ราคา", "เท่าไหร่"
→ Ask: รุ่นรถอะไรครับ?
→ Ask: ปีรถ?
→ Send recommended specs + price
→ Ask: สนใจนัดติดตั้งไหมครับ?
```

**Flow 3: Booking**
```
Trigger: Keyword "จอง", "นัด"
→ Show calendar available slots
→ Confirm booking
→ Send reminder 1 day before
```

## 📊 Crisis Management Checklist

### เมื่อเกิดปัญหา:

- [ ] ตอบกลับภายใน 30 นาที
- [ ] ขอโทษอย่างจริงใจ (ไม่แก้ตัว)
- [ ] เสนอวิธีแก้ชัดเจน
- [ ] กำหนด timeline
- [ ] ติดตามจนจบ
- [ ] ชดเชยเหมาะสม
- [ ] บันทึกเป็นบทเรียน
- [ ] ป้องกันไม่ให้เกิดซ้ำ

### หลังแก้ปัญหา:

- [ ] ขอ feedback จากลูกค้า
- [ ] อัปเดตกระบวนการทำงาน
- [ ] ฝึกอบรมทีม (ถ้าจำเป็น)
- [ ] ติดตามผลระยะยาว
