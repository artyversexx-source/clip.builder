# MODULE 5: 10 HIGH-CONVERSION CHAT CLOSING & UPSELL SCRIPTS

## สคริปต์ปิดการขายและ upsell สำหรับ "จีจี้ ล้อซิ่ง"

## 💬 ส่วนที่ 1: 5 Closing Scripts (ปิดการขาย)

### Script 1: The Price Drop (โปรโมชั่นจำกัด)
**สถานการณ์:** ลูกค้าสนใจแต่ยังลังเลเรื่องราคา

**บทสนทนา:**
```
ลูกค้า: ราคาเท่าไหร่ครับ?

พนักงาน: ชุดนี้สเปกเทพโปรโมชั่นจำกัด 3 ชุดสุดท้ายครับพี่! 
        ล้อลายฮิตขอบ 18 คู่ยางซิ่ง จบที่ราคาพิเศษสุดๆ 
        มัดจำจองคิววันนี้สลับล้อถ่วงล้อฟรี ทักเลยครับ! 🔥

[ถ้าลูกค้ายังกังวล]

พนักงาน: ปกติชุดนี้ 15,900 บาท แต่โปรนี้เหลือแค่ 12,500 บาทเท่านั้นครับ
        ประหยัดไป 3,400 บาท! แถมฟรีบริการติดตั้งอีก 1,500 บาท
        รวมประหยัดเกือบ 5,000 บาทเลยครับพี่!
```

**เทคนิค:**
- เน้นจำนวนจำกัด (Scarcity)
- แสดงมูลค่าที่ประหยัดได้
- เพิ่มของแถมที่มีค่าแต่ต้นทุนต่ำ

**อัตราการปิด:** 60-70%

---

### Script 2: The Sunday Urgency (เร่งด่วนวันอาทิตย์)
**สถานการณ์:** ลูกค้าทำงานธรรมดา ไม่มีเวลา

**บทสนทนา:**
```
ลูกค้า: ผมทำงานจันทร์-เสาร์ มาไม่ได้ครับ

พนักงาน: พี่ทำงานจันทร์-เสาร์ใช่ไหมครับ? ไม่ต้องลางานมา!
        วันอาทิตย์ร้านจีจี้เปิดเต็มระบบพร้อมลั่นครับ 
        ล็อกคิวช่อง VIP วันอาทิตย์นี้ไว้เลยไหมครับพี่?

[ถ้าลูกค้าสนใจ]

พนักงาน: วันอาทิตย์เราเปิด 9:00-18:00 ครับ
        ช่างพร้อม อุปกรณ์ครบ เหมือนวันปกติ
        แถมคนน้อยกว่า ได้บริการดุจดูแลมากขึ้นอีก!
        พี่สะดวกกี่โมงครับ เดี๋ยวผมจองคิวให้เลย
```

**เทคนิค:**
- แก้ปัญหาโดยตรง (ไม่มีเวลา)
- สร้างความพิเศษ (VIP queue)
- เสนอเวลานัดชัดเจน

**อัตราการปิด:** 50-60%

### Script 3: The Trade-In Bait (รับเทิร์นของเก่า)
**สถานการณ์:** ลูกค้ามีล้อเก่าอยู่แล้ว

**บทสนทนา:**
```
ลูกค้า: มีล้อเก่าอยู่ชุดหนึ่ง อยากเปลี่ยน

พนักงาน: เอารถเข้ามาเทิร์นล้อเดิมได้เลยครับพี่! 
        ช่างจีจี้ตีราคาประเมินหน้างานให้สูงที่สุด 
        เอาไปหักลบค่าล้อซิ่งชุดใหม่ จ่ายหน้างานเบาๆ สบายกระเป๋าครับ

[สอบถามเพิ่มเติม]

พนักงาน: พี่ใช้ล้อขอบอะไรอยู่ครับ? 
        ถ้าเป็นขอบ 15-16 เดิมโรงงาน เราเทิร์นให้ 2,000-2,500 บาทเลยครับ
        บวกกับโปรโมชั่นเดือนนี้ พี่อาจจ่ายเพิ่มแค่ 8-9 พันก็ได้!
```

**เทคนิค:**
- ลด pain point (ต้องจัดการของเก่า)
- ทำให้การจ่ายดูเบาลง
- กระตุ้นให้อยากประเมินราคา

**อัตราการปิด:** 70-80%

### Script 4: The Distance Killer (ลูกค้าต่างจังหวัด)
**สถานการณ์:** ลูกค้าอยู่ไกล กังวลเรื่องการติดตั้ง

**บทสนทนา:**
```
ลูกค้า: อยู่ต่างจังหวัด จะทำยังไง?

พนักงาน: อยู่ต่างจังหวัดห่วงเรื่องสลับล้อ? 
        ร้านจีจี้ถ่วงล้อไฮสปีด เติมลม พร้อมแพ็กหนาพิเศษ 
        ส่งฟรีถึงหน้าบ้าน ไปถึงอู่แถวบ้านขันน็อตใส่ จบหล่อทันทีครับพี่!

[อธิบายกระบวนการ]

พนักงาน: ขั้นตอนง่ายๆ ครับ:
        1. พี่เลือกสเปกที่เราแนะนำ
        2. เราถ่วงล้อ+เติมลมไนโตรเจนให้เสร็จ
        3. แพ็กกล่องกันกระแทกหนาพิเศษ
        4. ส่ง Flash Express 2-3 วันถึง
        5. พี่เอาไปให้อู่ใกล้บ้านขันน็อต (ค่าแรง 200-300 บาท)
        
        ง่ายมากครับพี่! ลูกค้าเราส่งทั่วประเทศทุกวัน
```

**เทคนิค:**
- แก้ทุกข้อกังวลล่วงหน้า
- ทำให้กระบวนการดูง่าย
- ยกตัวอย่างความสำเร็จ

**อัตราการปิด:** 40-50% (แต่ยอดต่อบิลสูง)

### Script 5: The Safe Fitment (รับประกันฟิตเมนท์)
**สถานการณ์:** ลูกค้ากลัวติดซุ้ม ล้อยื่น

**บทสนทนา:**
```
ลูกค้า: กลัวใส่แล้วติดซุ้ม/ล้อยื่นครับ

พนักงาน: รุ่นนี้ช่างร้านจีจี้จัดทรงมาแล้วเกิน 100 ลำครับพี่! 
        รับประกันฟิตเมนท์เป๊ะ ไม่ยื่นเกินบังโคลน 
        ไม่ติดซุ้มตอนเลี้ยวแน่นอน 
        พี่แวะเข้ามาลองทาบลายล้อก่อนได้เลยครับ

[สร้างความมั่นใจ]

พนักงาน: เรามีรถ demo ให้พี่ดูด้วยครับ 
        รุ่นเดียวกัน สีเดียวกัน ใส่แล้วทรงยังไง
        ถ้าพี่ไม่มั่นใจ เราวัดระยะให้ดูสดๆ เลย
        ใช้เครื่องมือวัด offset แม่นยำ 100%
        
        ถ้าใส่แล้วมีปัญหา เรายินดีแก้ไขให้ฟรีจนกว่าจะพอใจครับ!
```

**เทคนิค:**
- แสดงประสบการณ์ (100+ คัน)
- เสนอหลักฐาน (รถ demo)
- รับประกันผลลัพธ์

**อัตราการปิด:** 80-90%

## 🎯 ส่วนที่ 2: 5 Upsell Scripts (ขายเพิ่ม)

### Script 6: Lowering Springs Upsell
**สถานการณ์:** ลูกค้าซื้อล้อใหม่แล้ว

**บทสนทนา:**
```
พนักงาน: ได้ล้อแม็กใหม่ไปแล้ว ถ้าไม่ลดช่องว่างซุ้มล้อ 
        รถจะดูลอยครับพี่ 😅

        จีจี้แนะนำใส่สปริงโหลดตรงรุ่นเพิ่มอีกชุด 
        จัดทรงเตี้ยกำลังสวย นุ่มหนึบ ไม่ท้ายห้อยแน่นอนครับ

[อธิบายประโยชน์]

พนักงาน: สปริงโหลดของเรา:
        - ลดความสูง 30-40mm กำลังดี
        - นุ่มกว่าสปริงเดิมเล็กน้อย
        - ไม่กระทบการใช้งานประจำวัน
        - ราคาเพียง 8,500 บาท/คู่ (รวมติดตั้ง)
        
        เอาคู่กับล้อใหม่วันนี้ ลดพิเศษเหลือ 7,500 บาทครับ!
```

**เทคนิค:**
- ชี้ปัญหาที่ลูกค้าอาจไม่รู้
- เสนอวิธีแก้ที่สมบูรณ์
- ให้ส่วนลดเมื่อซื้อคู่

**อัตราการซื้อเพิ่ม:** 40-50%

---

### Script 7: Premium Lugs Cross-sell
**สถานการณ์:** ลูกค้าซื้อล้อสีพิเศษ

**บทสนทนา:**
```
พนักงาน: ล้อแม็กสีทองโครเมียมสวยขนาดนี้ 
        ใส่กับน็อตล้อเดิมโรงงานสีดำจะดูหมองครับพี่ 😅

        จัดน็อตล้ออะลูมิเนียมเกรดฟอร์จสีเข้าชุด 
        เพิ่มความเด่นให้ก้านแม็กอีกเท่าตัวครับ

[แสดงตัวเลือก]

พนักงาน: มีหลายสีครับ:
        - สีทอง (เข้ากับล้อทอง)
        - สีเงิน (คลาสสิก)
        - สีฟ้า/แดง/ดำ (สปอร์ต)
        
        ราคาเพียง 1,200 บาท/ชุด (4 ล้อ)
        เปลี่ยนแล้วรถดูพรีเมียมขึ้นทันที!
```

**เทคนิค:**
- ชี้จุดที่ปรับปรุงได้
- แสดงตัวเลือกหลากหลาย
- ราคาไม่สูง ตัดสินใจง่าย

**อัตราการซื้อเพิ่ม:** 60-70%

### Script 8: Hub Ring Safety Upsell
**สถานการณ์:** ลูกค้าซื้อล้อแฟชั่นรูใหญ่

**บทสนทนา:**
```
พนักงาน: ล้อแฟชั่นรูกลางจะกว้างกว่าดุมรถเดิมครับพี่ 
        ถ้าไม่ใส่ปลอกกันสั่น (Hub Ring) 
        เวลาขับเร็วเกิน 100 พวงมาลัยจะสั่นสู้มือ 😰

        จัดเพิ่มคู่ละไม่กี่ร้อย ขับขี่ปลอดภัยนิ่งๆ เลยครับ

[อธิบาย重要性]

พนักงาน: Hub Ring ทำหน้าที่:
        - ศูนย์กลางล้อให้ตรงกับดุม
        - ลดการสั่นสะเทือน
        - ยืดอายุลูกปืนล้อ
        
        ราคาเพียง 400-800 บาท/คู่ (แล้วแต่รุ่น)
        ลงทุนน้อย แต่สำคัญมากครับ!
```

**เทคนิค:**
- เตือนภัยความปลอดภัย
- อธิบายหน้าที่ชัดเจน
- ราคาต่ำ เมื่อเทียบกับความปลอดภัย

**อัตราการซื้อเพิ่ม:** 80-90% (ควรบังคับขาย)

### Script 9: Nitrogen Gas Cross-sell
**สถานการณ์:** ลูกค้าเปลี่ยนยางใหม่

**บทสนทนา:**
```
พนักงาน: เปลี่ยนยางใหม่ทั้งที แนะนำอัปเกรดเป็นเติมลมไนโตรเจนแท้ครับพี่ 
        ยางไม่ร้อน แรงดันเสถียร วิ่งนุ่มขึ้น 
        แถมสลับยางเติมฟรีตลอดอายุการใช้งาน

[เปรียบเทียบ]

พนักงาน: เติมลมปกติ vs ไนโตรเจน:
        
        ปกติ: ฟรี แต่ลมออกเร็ว ต้องเติมบ่อย
        ไนโตรเจน: 500 บาท/4 ล้อ 
                 - ลมอยู่ได้นาน 3-4 เท่า
                 - ยางไม่ร้อน วิ่งทางไกลปลอดภัย
                 - ประหยัดน้ำมันขึ้น 2-3%
                 - เติมฟรีตลอดชีพ!
        
        คุ้มค่ามากครับพี่!
```

**เทคนิค:**
- เปรียบเทียบให้เห็นชัด
- เน้นผลประโยชน์ระยะยาว
- บริการต่อเนื่อง (เติมฟรี)

**อัตราการซื้อเพิ่ม:** 50-60%

### Script 10: The Deposit Lock (มัดจำจอง)
**สถานการณ์:** ลูกค้าอยากได้แต่ยังไม่พร้อมจ่ายเต็ม

**บทสนทนา:**
```
พนักงาน: มัดจำเพียง 1,000 บาท 
        จีจี้ล็อกของชิ้นนี้จากโกดังให้ทันทีครับพี่! 
        ส่วนที่เหลือแวะมาจ่ายหน้างานหรือผ่อน 0% 
        วันติดตั้งได้เลยครับ 
        พี่สะดวกโอนช่องทางไหนดีครับ?

[ถ้าลูกค้ายังกังวล]

พนักงาน: มัดจำนี้พี่ได้สิทธิ์:
        ✅ ล็อกสินค้าไม่ให้คนอื่นซื้อ
        ✅ ล็อกโปรโมชั่นปัจจุบัน (แม้ราคาขึ้น)
        ✅ จองคิวติดตั้งล่วงหน้า
        ✅ ยกเลิกได้ หากราคาไม่ผ่านธนาคาร
        
        ไม่มีความเสี่ยงเลยครับพี่!
```

**เทคนิค:**
- ทำให้การตัดสินใจง่ายขึ้น
- เน้นสิทธิ์ที่ได้จากการมัดจำ
- ลดความเสี่ยง (ยกเลิกได้)

**อัตราการมัดจำ:** 70-80%

## 📊 เทคนิคการใช้สคริปต์ให้ได้ผล

### 1. ปรับใช้ตามสถานการณ์
- อย่าท่องจำแบบตายตัว
- ฟังลูกค้าก่อน ตอบโจทย์ตรงจุด
- ปรับภาษาให้เหมาะกับวัยลูกค้า

### 2. Timing สำคัญ
- เสนอ upsell หลังลูกค้าตัดสินใจซื้อแล้ว
- อย่าเสนอหลายอย่างพร้อมกัน (เลือก 1-2 อย่างที่สำคัญที่สุด)
- รอจังหวะลูกค้าผ่อนคลาย

### 3. ใช้ข้อมูลสนับสนุน
- แสดงรูป/วิดีโอประกอบ
- อ้างอิงลูกค้าเก่าที่ประสบความสำเร็จ
- มีตัวเลขชัดเจน (ประหยัดได้เท่าไหร่)

### 4. ปิดด้วยการกระทำ
- ทุกสคริปต์ต้องมี CTA ชัดเจน
- เสนอช่องทางการจ่ายเงิน
- นัดหมายเวลาที่ชัดเจน

## 🎯 Flow การคุยแนะนำ

```
1. ทักทาย + ถามความต้องการ
         ↓
2. แนะนำสินค้าที่เหมาะสม
         ↓
3. แก้ข้อกังวล (ถ้ามี)
         ↓
4. ปิดการขาย (Closing Script)
         ↓
5. เสนอ Upsell (1-2 อย่าง)
         ↓
6. ยืนยันออเดอร์ + นัดหมาย
```
